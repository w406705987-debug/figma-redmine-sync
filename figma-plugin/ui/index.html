🚀 第一步：安装后端服务

后端服务用于连接易协作，安装后会开机自动运行

1 点击下载后端工具
2 解压到任意目录
3 双击 install.bat 完成安装
⏳ 等待后端连接...

🔐 第二步：配置易协作认证

从浏览器获取 Cookie，自动拉取你名下所有未关闭的工单

1 请打开你的易协作网站并登录（例如：x20.pm.netease.com）
2 按 F12 打开开发者工具
3 点击 Application 标签页（在顶部更多页签中可打开）
4 选择左侧 Cookies 下的项目地址（例如：https://x20.pm.netease.com）
5 找到 name 列的 _my_redmine，并复制其 Value
🔒 配置仅保存在本地，自动拉取所有项目中指派给你的工单

🔐 修改易协作认证

1 请打开你的易协作网站并登录（例如：x20.pm.netease.com）
2 按 F12 → Application → Cookies
3 复制 _my_redmine 的值
