# Figma-易协作同步工具 v1.4.0 发布说明

## ✨ 新增功能

### 🎯 工单筛选增强
- **多选筛选**：支持同时选择多个工单状态进行筛选
  - 示例：可以同时查看「新建」和「UI设计中」的工单
  - 不再局限于单一状态筛选
  
- **状态记忆**：插件自动记住上次的筛选选择
  - 关闭插件后重新打开，筛选条件自动恢复
  - 提升工作连续性和效率

## 🔧 技术改进

### 持久化存储优化
- 使用 Figma 官方 `clientStorage` API 替代 localStorage
- 解决 iframe 安全策略导致的存储失败问题
- 数据存储更稳定、更安全

### UI 交互优化
- 筛选面板支持 checkbox 多选
- 选中状态视觉反馈更清晰
- 筛选逻辑支持复杂组合条件

## 📦 安装说明

### 方式一：在线安装（推荐）
1. 下载本 Release 中的 `backend.zip`
2. 解压到任意目录
3. 双击运行 `start.bat` 启动后端服务
4. 在 Figma 中打开插件即可使用

### 方式二：手动安装
```bash
# 克隆仓库
git clone https://github.com/w406705987-debug/figma-redmine-sync.git

# 进入后端目录
cd figma-redmine-sync/backend

# 安装依赖
npm install

# 启动服务
npm start
```

## 📝 使用说明

### 多选筛选使用方法
1. 点击「我的工单」页签
2. 点击顶部「筛选」按钮
3. 勾选多个状态（如「新建」+「UI设计中」）
4. 工单列表自动更新，显示所有选中状态的工单
5. 关闭插件后重新打开，筛选条件自动恢复
